# Проект: собственный DNS-резолвер с внешним кэшем и обходом ограничений DNSSEC

## Цель проекта
Развернуть собственный DNS-резолвер, способный кэшировать DNS-ответы на время, отличное от TTL, с использованием внешнего кэша Redis, а также продемонстрировать локальную подмену DNSSEC-подписанной зоны.

- Week 1 — стенд, Unbound, DNSSEC-базис, Redis, test-plan и документационный каркас

- Week 3 — внешний Redis-кэш
- Week 4 — локальная подмена DNSSEC-зоны
