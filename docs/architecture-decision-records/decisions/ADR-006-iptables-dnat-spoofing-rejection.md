# ADR-006: Использование iptables DNAT для механизма подмены DNS-зон

## Статус
Отклонено

## Дата
2026-04-04

## Контекст
На четвертой неделе при реализации DNS-Spoofing (локального перехвата ответов к существующим доменам, например, к `isc.org`) рассматривался вариант перенаправления запросов не средствами конфигурации DNS-резолвера L7, а на сетевом уровне с помощью правил Netfilter/iptables. Идея заключалась в применении DNAT (Destination Network Address Translation) для прозрачного перехвата UDP-пакетов на порт 53 и их перенаправления на виртуальный локальный сервис-заглушку.

## Решение
Решение использовать iptables или nftables для подмены (spoofing) DNS-запросов отклонено в пользу использования встроенных директив `local-zone` внутри Unbound.

## Обоснование
- **Сложность L7 (уровень полезной нагрузки):** `iptables` оперирует на уровнях L3/L4 (IP/UDP/TCP) и не умеет взаимодействовать со структурой самого DNS-сообщения. Сетевой перехват пакета потребует наличия дополнительного фейкового DNS-сервера на принимающей стороне для генерации легитимного пакета с нужным адресом (например, `192.0.2.0`) и нужными флагами (AA/AD). 
- **Логика DNSSEC:** При подмене на уровне сети оригинальный ответ от авторитетного сервера не дойдет до валидатора, либо ответ от сервис-заглушки не пройдет проверку DNSSEC. Управление обходом проверок (инвалидацией) для подмененных доменов технически невозможно реализовать на уровне iptables, это должно производиться внутри валидатора резолвера (параметр `domain-insecure`).
- **Модификации TTL:** Одна из задач 4-й недели предполагала динамическую модификацию TTL (`respect`, `override`, `min`). Эти манипуляции с полями протокола DNS невозможно реализовать через DNAT.

## Последствия
- Выбран механизм `local-zone` внутри Unbound (как закреплено в ADR-007). 
- Ответственность за подмену лежит исключительно на приложении, что делает конфигурацию более портативной (работает внутри Docker без необходимости дополнительных флагов привилегированных сетевых возможностей `NET_ADMIN` для контейнера).
